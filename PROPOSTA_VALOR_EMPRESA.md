# 💰 PROPOSTA DE VALOR - H.Olhos Renomeador de Guias

## Executive Summary

**H.Olhos** é uma solução de automação que elimina o processo manual de renomeação de guias médicas, economizando **15-25 horas por mês** por operador e reduzindo erros em **95%**.

---

## 🎯 PROBLEMA ATUAL (Diagnóstico)

### Processo Manual Hoje
```
Tempo por Guia:      2-3 minutos
├─ Abrir arquivo.... 20s
├─ Extrair dados.... 60s
├─ Renomear......... 40s
└─ Salvar........... 20s

Guias por dia:       200-300 arquivos/dia
Operador Tempo:      6-15 horas/dia processando

Erros Manual:        5-10% de guias com nome errado
├─ Nomes digitados errado
├─ Números missing
├─ Caracteres inválidos
└─ Duplicatas
```

### Custos Atuais (por operador/mês)
```
Salário operador........... R$ 2.500/mês
├─ 160h/mês trabalhadas
├─ R$ 15,63/hora

Tempo em renomeação:
├─ 50% do tempo = 80h/mês
└─ Custo = R$ 1.250/mês APENAS nessa tarefa

Erros (rework):
├─ 5-10% de arquivos errados
├─ Tempo para reparar = 10h/mês
├─ Custo rework = R$ 157/mês

TOTAL CUSTO MENSAL: R$ 1.407/operador
ANUAL (3 operadores): R$ 50.652
```

## ✨ SOLUÇÃO: H.OLHOS

### Como Funciona
```
1. Upload do PDF/Imagem
   ├─ Extração automática de nome ✅
   ├─ Extração automática de guia ✅
   └─ OCR para documentos escaneados ✅

2. Processamento Inteligente
   ├─ Reconhecimento de padrões (AI-assisted)
   ├─ Normalização de dados
   └─ Validação automática

3. Geração de Nome
   ├─ Formato padrão: [Nome]_[GuiaNumber]
   ├─ Remoção de caracteres inválidos
   └─ Preenchimento de zeros à esquerda

4. Download em Lote
   ├─ ZIP com todos os arquivos renomeados
   └─ Geração em segundos
```

### Resultados Mensuráveis

Throughput/dia          200-300   1000+      3-5x ↑
Horas por 1000 guias    33-50h    5-8h       75% ↓

**Solução:** Interface permite review e correção antes de salvar

### Risco: Falha Técnica

- Arquivo original sempre preservado
